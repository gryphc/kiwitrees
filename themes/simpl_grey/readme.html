for kiwi-webtrees

is the theme I use on my personal kiwi-webtrees site Our Families. It is unusual in having a fixed width, very few icons, and a totally unique (for kiwi-webtrees) menu structure. Its overall look is simple, and has some similarities to Apple's websites.
This module requires kiwi-webtrees version .
Support for Simpl_add-ons is available only at the Kiwitrees forum pages.

Updates and bug-fixes will be announced on the Simpl_add-ons blog pages, and on Twitter.

Installation is simply a matter of unzipping the downloaded zip file and copying the unzipped "" folder directly into your kiwi-webtrees/themes/ folder.

It will then appear as an option in the theme selection drop-down menu.

No additional setup is required unless you wish to make this the default site of your site, family tree, or user account. You can set these in the kiwi-webtrees administration pages Site configuration; Family treesyour family tree; or any user account details.
To avoid any complication with possible file removals between versions, we recommend that you completely delete your old version of the theme before replacing it with this new one.
This theme includes some text that is not part of standard kiwi-webtrees. That means it will not normally be translated from its default US_English. If you require translation of these items you will need to use kiwi-webtrees "Custom translation" feature described in the kiwi-webtrees WIKI here:Custom Translation. You might also want to change some existing kiwi-webtrees translations that are too long for the Simpl_grey menu design. For example, in webtrees 1.2.7 the menu item "FAQ" was translated in French to "Questions frequentes (FAQ)" which is too long to fit the space designed.
